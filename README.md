# vue-starter

## VS Code Plugins

### Must
- Vetur (Vue tooling)
- ESLint
- Prettier
- Babel ES6/ES7

### Optionals
- Vue Peek (goto defs)
- Auto Close Tag
- Auto Rename Tag
- Path Intellisense
- EditorConfig


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```
